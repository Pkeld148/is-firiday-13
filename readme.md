# is-Friday-13

## When you need to know if it's Friday the 13th fast and reliably.



## Installation
```
npm install is-friday-13
```

## Usage
```
const isFriday13 = require("is-friday-13");
const spookyCheck = isFriday13();

console.log(spookyCheck);
```
